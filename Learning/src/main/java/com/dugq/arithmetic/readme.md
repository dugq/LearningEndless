# 二分法注意点
* 当节点数为偶数时，中点有两个，可以选择(left+right)/2 和 （left+right)/2 选择
* 当节点数为偶数时，无论怎么选择，中点都不可能遍历整个数组。（要么会忽略起点，要么会忽略中点）
  * 所以必须先对起点或者重点进行判断是否符合条件
